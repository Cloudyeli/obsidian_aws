Command Line Interface (CLI)

### [[EC2]] CLI Command
#command #cli #ec2

##### - Launch [[EC2]] instance
	aws ec2 run-instances --image-id ami-xxxxxxxx --count 1 --instance-type t2.micro

### [[S3]] CLI Command
#command #cli #s3
##### - list all s3 buckets
	aws s3 ls
##### - List the content of a storage container (bucket) on Amazon [[S3]]
	aws s3 ls s3://mys3databucket
