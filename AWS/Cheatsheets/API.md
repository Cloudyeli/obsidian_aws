### AWS API [[Pricing]]

*   Query the prices of AWS services  
*   **Price List Service API (AKA the Query API)** – query with JSON
*   **AWS Price List API (AKA the Bulk API)** – query with HTML
*   Alerts via Amazon SNS when prices change
