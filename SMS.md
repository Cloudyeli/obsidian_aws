AWS ## Server Migration Service (SMS)

==**AWS Application Migration Service (AWS [[MGN]])**Â is the primary migration service recommended for lift-and-shift migrations to AWS. Customers currently using Server Migration Service (SMS) are encouraged to switch to Application Migration Service for future migrations.

AWS Server Migration Service (SMS) is an agentless service which makes it easier and faster for you to migrate thousands of on-premises workloads to AWS
AWS SMS allows you to automate, schedule, and track incremental replications of live server volumes, making it easier for you to coordinate large-scale server migrations
